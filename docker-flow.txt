- Create backend container
    - Install python dependencies
    - Create superuser from cli
    - Load existing database? y/n
        If yes -> point to database location and load
    - Seed data from db? y/n
        If yes --> seed data and use create superuser for recipes
    - Install ollama
        - Ask which model to use as default
    - Install ffmpeg
    - Ask for specific vosk model (if wanted) save in backend/recipes/extraction/utils
        - Else install: https://alphacephei.com/vosk/models/vosk-model-en-us-0.22-lgraph.zip


- Create frontend container
    - Install npm dependencies

- Configure Keycloak
    - Create realm and public client for the web app
    - Set valid redirect URIs to http://localhost:4200/* and silent-check-sso.html
    - Expose realm OIDC issuer and JWKS endpoints
    - Provide .env values for backend verification

- Choose auth provider
    - Local dev uses JWT by default (ng serve development)
    - To run with Keycloak locally: npm run start:keycloak
    - Production can be built with either provider by editing environment.prod.ts (authProvider)

- Run hehe
