# emma-cookbook Frontend

React/Vite frontend for the emma-cookbook application. EMMA stands for Easy Meals Made Accessible.

## Local development

Requirements:

- Node.js 20+
- npm

Run locally:

```sh
cd cookbook-app
npm install
npm run dev
```

Build for production:

```sh
cd cookbook-app
npm run build
```

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Notes

- The current frontend still uses local/mock state for most domain data.
- Backend integration and schema alignment are documented in `docs/`.
